Getting Started

Convert your Jupyter Notebooks (.ipynb) or Python scripts (.py) into professionally formatted documents in just a few clicks.

Upload Methods

  • File Upload: Drag and drop or click to upload files from your computer
  • URL Import: Paste a link from Google Colab, GitHub, or any direct .ipynb URL

Supported Sources

  • Google Colab: colab.research.google.com/drive/...
  • GitHub Notebooks: github.com/user/repo/blob/.../notebook.ipynb
  • GitHub Gists: gist.github.com/user/gist_id
  • Direct URLs: Any URL ending with .ipynb

Note: For Google Drive files, ensure the file is publicly shared (Anyone with the link).

Export Formats

  • DOCX: Microsoft Word format - ideal for editing and academic submissions
  • HTML: Web-ready format - perfect for sharing online or printing

Export Options

  • Templates: Choose from Academic Report, Technical Documentation, or Basic styles
  • Line Numbers: Display line numbers in code blocks
  • Include Outputs: Include cell execution outputs in the document
  • Table of Contents: Auto-generate TOC from markdown headings
  • Page Numbers: Add page numbers to the document

Features

  • Syntax highlighting for code cells
  • Automatic figure extraction and captioning
  • Markdown rendering with proper formatting
  • Support for LaTeX equations
  • Responsive preview before conversion

About This Project

Smart Notebook Print Formatter

Version 0.1.0

A personal tool designed to streamline the process of converting Jupyter Notebooks and Python scripts into professionally formatted, print-ready documents.

Created By

Tanmay Shinde

This tool was built for academic use to save time when preparing notebook submissions, lab reports, and project documentation.

Purpose

As a student, I often found myself spending too much time formatting notebooks for submission. This tool automates that process, allowing me to focus on what matters most - the actual work.

  • Quick conversion of notebooks to DOCX/HTML
  • Consistent, professional formatting
  • No more manual copy-pasting into Word
  • Preserves code highlighting and outputs

Python FastAPI JavaScript HTML/CSS nbformat python-docx
